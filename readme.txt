This project was born out of need to work with Admssion Data of applicant into one of Nigeria public University
This Python script generate fake University data for each candidate,
It allocate Olevel Result, UTME result, POST UTME.
It calculate weighted score each candidate olevel, utme and post-utme and find the totat aggregate score.
It equally detrmine admission status of student based on calculated parameter for screen

The script work by user submit the following data as parameter
1. number of user you which to generate
2. programme applying for by the student
3, the cutoff_mark for such programme


Out result
an excel file will be genrated that will conating records of each student in row.
